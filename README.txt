Kõige lihtsam on antud Scala projekt käivitada, kui laadida alla sbt aadressilt http://www.scala-sbt.org/download.html. 
Seejärel minna käsureal lahtipakitud programmi kausta ja anda käsk “sbt run”. 
Teine võimalus on laadida alla IntelliJ IDEA, seal installeerida Scala plugin ja projekt avada antud IDE kaudu. 
Viimasel juhul tuleb käivitada fail Game.scala.
